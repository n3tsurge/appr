# AppR

An enterprise application inventory and management system built with Vue.js and Python FastAPI. AppR provides a comprehensive solution for tracking, managing, and optimizing enterprise applications, ensuring efficient resource utilization and streamlined operations.

## Features

- **Application Inventory**: Maintain a centralized inventory of all enterprise applications, including details such as name, description, type, owner, and associated resources.
- **Resource Management**: Track and manage resources associated with each application, including databases, servers, APIs, and more.
- **Incident Tracking**: Log and manage incidents related to applications, ensuring timely resolution and minimizing downtime.
- **User Management**: Manage users and their roles within the system, ensuring appropriate access control and security.
- **Search and Filtering**: Easily search and filter applications and resources based on various criteria, making it simple to find the information you need.
- **Reporting and Analytics**: Generate reports and analytics to gain insights into application usage, performance, and resource allocation.
- **Scorecarding and Health Metrics**: Monitor the health and performance of applications using customizable scorecards and metrics.  Readiness scorecards can be used to assess the preparedness of applications for deployment, while health scorecards provide ongoing insights into application performance and stability.  Security scorecards can evaluate the security posture of applications, identifying potential vulnerabilities and areas for improvement.
- **Integrations**: Integrate with external systems such as monitoring tools, incident management platforms, and authentication providers for a seamless experience.
- **API Access**: Provide a RESTful API for programmatic access to application inventory data, enabling integration with other tools and systems.

## Technology Stack

- **Frontend**: Vue.js, Vue Router, Vuex, Axios, Pinia, Tailwind CSS
- **Backend**: Python FastAPI, SQLAlchemy, PostgreSQL
- **Authentication**: JWT (JSON Web Tokens), OAuth2, SAML
- **Deployment**: Docker, Kubernetes
- **CI/CD**: GitHub Actions
